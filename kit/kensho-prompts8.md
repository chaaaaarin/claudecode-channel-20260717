# 🎁 新モデル検証プロンプト集（8問）

新しいAIモデルが出るたびに同じプロンプトで試せば、横並びで実力を比較できる検証用プロンプト集です。

**受け取り方**: 各コードブロックのコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: 新しいモデルを試すたびに同じ8問を投げて、結果をメモしておくと横比較しやすくなります。

## 状態管理・インタラクション

### 1. 群衆シミュレーション
説明: 複数主体の状態管理・並行処理の実力を見る。
```text
3つのエレベーターがある10階建てビルのシミュレーションを作って。各階から人がランダムに現れ、行き先階を指定して呼び出しボタンを押す。エレベーターは同時に1人しか運べず、乗れなかった人は次のエレベーターを待つ。各人にマウスホバーすると行き先階がツールチップで表示されるようにして。
```

### 2. 3Dインタラクション（開閉ギミック）
説明: 3Dオブジェクトとクリック操作の連動を見る。多くのモデルが苦手とする難問。
```text
コンタクトレンズケースの3Dモデルを作って。LとRの蓋がはっきり見分けられるようにラベルを付け、クリックすると蓋が開閉するアニメーションにして。
```

### 3. 折りたたみ3Dアニメーション
説明: スライダー操作と3Dアニメーションの同期精度を見る。
```text
3Dの折りたたみテーブルを作って。スライダーを右にドラッグすると天板が展開し、左にドラッグすると折りたたまれる、なめらかなアニメーションにして。
```

## 描画・ビジュアル

### 4. SVGキャラクター描画
説明: プロポーションと状況説明力を見る定番テスト。
```text
ハンバーガーを食べているパンダのSVGイラストを作って。パンダらしいプロポーションと、ハンバーガーを美味しそうに食べている様子が伝わるように。
```

### 5. ゲーム物理演算
説明: 物理挙動の実装力とゲームロジックの組み合わせを見る。
```text
弓矢で的を射るゲームを作って。的は4つ、一番早く全部命中させた人がリーダーボードに載る。矢の物理挙動（重力・軌道）をリアルにして。
```

## 推論・長時間タスク

### 6. 難問の数学的推論
説明: 単純な暗記では解けない組み合わせ問題で、地力の推論力を見る。答えは自分で検算すること。
```text
1から9までの数字から重複なく3つ選び、a<b<cかつa+b+c=15を満たす組み合わせ(a,b,c)を全部数えて。答えの個数だけ教えて、根拠は簡潔に。
```

### 7. 長時間自律タスク
説明: エラーが出ても自力で最後までやり切る粘り強さを見る。時間がかかるので余裕があるときに。
```text
ある動物についての面白い事実を50個集めたデータセットを作って。それを使って小さな言語モデルをファインチューニングし、リロードのたびに新しい事実をランダム表示するローカルWebページを作って。エラーが出ても自力で直しながら最後まで終わらせて。
```

### 8. 3D精密再現（最難関）
説明: 現状どのモデルも完全クリアが難しい最難関の複合3Dタスク。伸びしろの確認に。
```text
現在時刻を正確に表示する3D腕時計を作って。秒針・分針・時針がなめらかに動き、日付表示と2つのタイムゾーンを切り替えられるようにして。
```

以上、8問です。内訳は状態管理・インタラクション3問、描画・ビジュアル2問、推論・長時間タスク3問。

**ポイント**: 数学問題（6問目）は答えを自分で検算してから採点してください。モデルが正解を出しても途中式が間違っていることがあります。
